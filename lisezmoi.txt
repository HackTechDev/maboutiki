Installation 

1) Créer le répertoire ~/IMAUGIS

2) Installer Xampp for Linux v1.8.1 dans le répertoire ~/IMAUGIS/

 http://www.apachefriends.org/fr/xampp-linux.html

3) Créer le répertoire maboutiki/ : ~/IMAUGIS/lampp/htdocs/maboutiki/

4) Copier les fichiers dans le répertoire : ~/IMAUGIS/lampp/htdocs/maboutiki/

5) Créer le répertoire ~/IMAUGIS/lampp/htdocs/sites
   Mettre les permissions 777

6) Lancer la création du site test : http://localhost/maboutiki/installerSite.php

7) Vérification : 
   ~/IMAUGIS/lampp/htdocs/sites/test
   et avec phpMyadmin : http://localhost/phpmyadmin/

8) Site de test : http://localhost/sites/test/
